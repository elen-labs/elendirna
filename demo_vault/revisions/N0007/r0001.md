## Delta

Elendirna는 에이전트 친화적인 지식 베이스 CLI 도구입니다.

### 핵심 개념
- **사람과 AI의 공유 기억**: 컨텍스트 윈도우 한계와 사람의 기억력 한계를 보완하기 위한 도구 (회의록 역할).
- **ELF 컨벤션 내재화**: 기존 문서 기반 규칙을 CLI 코드에 강제력 있는 형태로 캡슐화하여 AI의 토큰 낭비와 할루시네이션 방지.

### 주요 철학
1. **Base-Delta**: 기준(baseline) 대비 변경분만 기록.
2. **Identifier / Attribute 분리**: 파일명은 식별자용, 메타데이터는 manifest에 저장.
3. **SSoT (Single Source of Truth)**: 단일 vault에 모든 정보를 공존.
4. **Protocol-first**: 도구 없이도 사람이 직접 읽을 수 있는 데이터 구조 보장.
